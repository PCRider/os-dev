Каталог kernel содержит исходные коды ядра.
Каталог drivers содержит исходные коды драйверов.
Каталог programs содержит исходные коды программ, вошедших в дистрибутив.
  * Отсутствуют исходники следующих программ:
    * hdview
    * jmail
    * htmlv
    * padenie
Каталог lib содержит исходники библиотек, вошедших в дистрибутив.
Каталог Skins содержит исходники скинов. Если вы хотите создать свой скин,
посмотрите существующие примеры. Исходник стандартного скина
(WinXP_Standart) лежит в svn-репозитории вместе
с ядром и потому находится в папке kernel\skin. При создании
скина нужно указывать таблицу цветов (dtp-файл); возможно, вам понравится одна
из входящих в папку Skins\DTP.
Каталог other содержит исходники, не вошедшие ни в одну из других папок.
